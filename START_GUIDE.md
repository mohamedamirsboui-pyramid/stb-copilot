# 🚀 Comment lancer STB Copilot

Pour lancer votre projet, vous devez ouvrir **deux terminaux** différents (un pour le Backend, un pour le Frontend).

## 1. Lancer le Backend (API)
Ouvrez un terminal et tapez :
```bash
cd /home/pyramid/stb-copilot/backend
dotnet run
```
> Le backend est prêt quand vous voyez : `Now listening on: http://localhost:5000`

---

## 2. Lancer le Frontend (Interface)
Ouvrez un **deuxième** terminal et tapez :
```bash
cd /home/pyramid/stb-copilot/frontend
npm start
```
> Le frontend est prêt quand vous voyez : `Local: http://localhost:4200/`

---

## 3. Accéder à l'application
Une fois que les deux sont lancés, ouvrez **Firefox** et allez sur :
👉 **[http://localhost:4200](http://localhost:4200)**
